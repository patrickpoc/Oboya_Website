# 17 — Do and Don't

## Objetivo

Listar apenas restrições e práticas **explícitas** (ou diretamente derivadas) do Brandbook 2026. Sem don'ts inventados.

## Cores

| Do | Don't |
|----|-------|
| Começar com Main Blue + Main Green | Ignorar as cores estruturais |
| Usar White ou Soft White como neutros | Introduzir outros neutros como “superfície de marca” oficial |
| Um accent (Light Yellow **ou** Orange Red), ≤ 25%, com intenção | Usar Light Yellow **e** Orange Red na mesma peça |
| Accent pontual | Accent como fundo ou grande superfície |
| Dark Yellow para texto que precisa destacar-se em fundo claro | Texto Light Yellow sobre branco |
| Dark Blue para texto corporativo / profundidade | Dark Blue como fundo primário em grandes formatos |
| Dark Blue / institucional em técnico, fichas, distribuidores, certificações | Conteúdo institucional Dark Blue em packaging ou social media |

## Logo

| Do | Don't |
|----|-------|
| Respeitar clear space = altura do “O” | Deixar elementos invadirem o clear space |
| Usar versões Light / Dark adequadas ao fundo | Distorcer, recolorir fora da paleta, ou inventar efeitos (não autorizado no texto) |

*Usos incorretos ilustrados no PDF sem texto:* **não informados** aqui até haver prints/descrições — não inventar lista.

## Tipografia

| Do | Don't |
|----|-------|
| Neue Haas Grotesk Display em títulos | Substituir a identidade tipográfica por fontes genéricas como padrão de marca |
| Plus Jakarta para subtítulo/corpo | Ignorar a hierarquia Title / Subtitle-Body / Tags |
| ±8px óptico com intenção | Alterar escala sem critério |

## Imagem

| Do | Don't |
|----|-------|
| Produtos frescos; embalagens impecáveis | Produtos murcho / embalagem danificada |
| Perspectiva alinhada em packshots frontais | Horizontais/verticais tortas em foto frontal |
| Overlay quando texto sobre foto/vídeo | Texto ilegível sobre imagem |
| Leaf/Circle para destacar imagem; Tag para texto sobre imagem | Shapes sem função |

## Social / feiras

| Do | Don't |
|----|-------|
| Respeitar safe zones (1080×…) | Cortar logo/CTA fora da safe zone |
| 80 cm livres na base do booth; QR visível | Bloquear a base ou esconder QR |

## Flexibilidade (oficial)

> Only the specified restrictions must be strictly adhered to; everything else is subject to change and interpretation based on the designers’ judgment and expertise.

**Do:** inovar dentro do sistema.  
**Don't:** violar as restrições da tabela acima.
